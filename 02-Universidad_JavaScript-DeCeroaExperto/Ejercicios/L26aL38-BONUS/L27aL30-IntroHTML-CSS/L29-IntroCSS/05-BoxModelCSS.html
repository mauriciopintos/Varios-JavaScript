Manejo de Box model con CSS

Modelo de caja CSS

Veremos el modelo de caja de CSS:
margin: Separacion entre elementos. No tiene otro atributo mas que distancia.
border: Lo que decora/adorna el elemento. Se encuentra entre el margin y el padding.
A demas de distancia, este puede tener color e intensidad.
padding: Separacion entre el nucleo del elemento y el borde. No tiene otro atributo mas que distancia.
