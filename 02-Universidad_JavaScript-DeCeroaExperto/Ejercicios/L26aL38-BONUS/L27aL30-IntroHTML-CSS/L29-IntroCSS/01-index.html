Manejo de colores con HTML

Color por nombre: crimson
https://htmlcolorcodes.com/color-names/

Color por valor hexadecimal: #dc143c

Color por valor RGB: rgb(220, 20, 60)

Color por valor HSL: hsl(348, 83%, 47%)

Borde color por valor RGBA (A= Alpha transparencia): rgba(220, 20, 60, 0.5)

Borde color por valor HSLA (A= Alpha transparencia): hsla(348, 83%, 47%, 0.5)"

En ambos casos la transparencoa va de 0 a 1, es decir con todos los decimales entre 0 y 1
